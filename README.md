# Data-Pill-iOS
App Challenge #05

## Description
Data Pill tracks device mobile data usage.

Features:
- Track daily data usage or total data usage for a period
- See weekly overview of data usage
- Set plan and daily data limit

Built with:
- Core Data, User Defaults (Local Storage)
- Swift UI 
- MVVM Design
